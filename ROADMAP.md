# Inkswell Roadmap

Phase-centric information architecture (one tab, left icon rail): **Home · Plan · Write · Track · Revise · Publish**. Each destination does one job; depth lives behind ≤3 sub-tabs or the shared Scene Inspector. See `FEATURES.md` for the full pick list and the plan file for rationale.

## Shipped
- **v0.1.0** Projects (Longform-compatible) + explorer + compile (md/HTML/pandoc)
- **v0.2.x** Goals + sprints + stats; scenes-vanishing fix; explorer toolbar
- **v0.3.0** Single-tab host view (internal tabs swap panels)
- **v0.4.0** Save the Cat beat sheet

## Out of scope (philosophy)
AI; cloud sync / real-time collaboration; replacing Obsidian's editor; separate database/account.
